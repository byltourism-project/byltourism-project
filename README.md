 BYL Tourism - Agência de Viagens

Projeto de um site de turismo moderno que apresenta destinos brasileiros focando em **experiências** e **gastronomia**.

Apresentados
* **Salvador (BA):** Cultura, Pelourinho e culinária baiana.
* **Aurora (TO):** Natureza exuberante e Rio Azuis.
* **Balneário Camboriú (SC):** Praias badaladas e gastronomia europeia.

Utilizadas
* **HTML5:** Estrutura das páginas e formulários.
* **CSS3:** Estilização com Flexbox, layouts responsivos e efeitos de hover.
* **YouTube Embed:** Integração de vídeos para imersão do usuário.
